# HobbyFlow App

React Native (Expo) mobile app for curated hobby learning.

## Scripts

- `npm start` — Expo dev server
- `npm run android` — run on Android
- `npm run web` — run on web

## Setup

```bash
cp .env.example .env
npm install
npm start
```

Point `EXPO_PUBLIC_API_URL` at your local `hobbyflow-server` (default `http://localhost:3000`).
